# CareerCopilot

CareerCopilot is a full-stack web application designed to help students and job seekers improve their career preparation. The platform allows users to upload resumes, evaluate ATS compatibility, identify skill gaps, generate personalized learning roadmaps, and prepare for interviews through a simple and intuitive interface.

---

## Features

- 🔐 Secure User Authentication (JWT)
- 📄 Resume Upload & PDF Parsing
- 📊 ATS Resume Score Analysis
- 🎯 Skill Gap Detection
- 🛣️ Personalized Career Roadmap
- 💬 Interview Preparation
- 👤 User Dashboard & Profile Management

## Getting Started

### Clone the repository

```bash
git clone https://github.com/moinak05-m/CareerCopilot-AI.git
```

### Install dependencies

```bash
npm install
```

### Start the project

Backend

```bash
cd backend
npm install
npm run dev
```

Frontend

```bash
cd frontend
npm install
npm run dev
```
